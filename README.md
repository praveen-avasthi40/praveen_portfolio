## 🧠 About This Project

This is my **personal portfolio website** built to showcase my:
- Skills
- Projects
- Technical expertise
- Development journey

Unlike traditional static portfolios, this project is designed to evolve into a **dynamic, animation-driven experience** inspired by modern UI/UX principles.

---

## 🎯 Objective

Most student portfolios fail because they:
- Look generic
- Have no interaction
- Don’t tell a story

This project solves that by focusing on:
✔ Clean UI & Working backend
✔ Structured content  
✔ Expandable animation architecture
